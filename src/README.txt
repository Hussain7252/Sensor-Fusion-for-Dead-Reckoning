The enitre Analysis code is written in Matlab and the codes are in the folder analysis under src.
1.) Hard and Soft Iron calibration code matlab file name is  Hardnsoftcalib.m (To successfully run this code you will require imucircle.mat and fit_ellipse.m file which is also available in the analysis folder

2.) The yaw calculation from Magnetometer and gyro and Complimentary filter yaw calculation code is written in the matlab file with name Part1_analysis_cont.m (To successfully run this code you will 
require imudriving.mat file which is also available in the analysis folder)

3.) Velocity calculation from Accelerometer code is written in the matlab file with name Part2_analysis.m (To successfully run this code you will 
require imudriving.mat and gpsdriving.mat file which is also available in the analysis folder)

4.) Trajectory estimation from IMU and GPS code is written in the matlab file with name Part3_analysis.m (To successfully run this code you will 
require imudriving.mat and gpsdriving.mat file which is also available in the analysis folder)
